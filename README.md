# Android Weather App
* Capturar Geolocalizacion(Latitude, Longitude)
* Obtener el nombre de la ciudad con Geocoder
* Buscar el clima de la ciudad consultando el servicio de OpenWeatherMap  (http://openweathermap.org/)
